# Atask: GitHub Repository Explorer

A minimal GitHub repository explorer built with Next.js, integrated with GitHub's GraphQL API and tested using Playwright.

## Getting Started

### Local Development

```bash
npm install
npm run dev
# Open your browser at http://localhost:3000
```

### Running Tests

```bash
npx playwright test
```
